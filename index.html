Biblioteca de Importações
Adicione, exclua e consulte as planilhas já integradas à base.
Ao importar, os registros são consolidados com a base atual. A exclusão de uma planilha remove as leituras vinculadas àquela importação.
Visualização rápida
Selecione uma planilha para abrir a prévia sem baixar.

Use “Escrever novo” quando o sistema, subunidade ou parâmetro ainda não existir na base importada.
Editar ou criar subunidade
Renomeie uma subunidade existente ou crie uma nova subunidade para receber leituras manuais.
